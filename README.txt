This program contains a collection of classes; the Country, Question, Quiz, and QuizResults classes consist of constructor and methods used to create the logic of a quiz. The Geography Quiz class contains the javafx basis of the quiz and pulls methods and objects from the other files to create the visuals and mechanics of a geography quiz.

     1) Compile the classes with the command:

     	mvn compile


     2) Run the classes with the command:

     	mvn -e -Dprism.order=sw exec:java -Dexec.mainClass="edu.uga.cs1302.quiz.GeographyQuiz"
     	

     3) Take the Quiz!
